## Remote add files

Become a hacker with `curl`and `git` only.

```bash
# tmux
git clone --depth 1 https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
curl -sSL https://github.com/thenets/dotfiles/raw/master/.tmux.conf > ~/.tmux.conf
```
